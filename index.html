ตั้งค่าการเชื่อมต่อ Google Sheets Live Sync

วาง URL ที่ได้จากการ Deploy Apps Script เป็น Web App (สิทธิ์เข้าถึง: Anyone) เพื่อดึงข้อมูลสดตรงจากชีตของคุณ

วิธีการตั้งค่าใน Google Sheet:

  1. เปิด Google Sheet ของคุณ
  2. เลือกเมนู ส่วนขยาย (Extensions) -> Apps Script
  3. ก๊อบปี้โค้ดในไฟล์ google_apps_script.js ไปวางแทนโค้ดเดิมทั้งหมดแล้วกดบันทึก
  4. กดปุ่ม การทำให้ใช้งานได้ (Deploy) -> การทำให้ใช้งานได้ใหม่ (New deployment)
  5. เลือกประเภทเป็น เว็บแอป (Web app), กำหนดผู้มีสิทธิ์เข้าถึงเป็น ทุกคน (Anyone)
  6. กด Deploy คัดลอก URL ที่ได้มาวางที่ช่องด้านบน แล้วกดบันทึก
